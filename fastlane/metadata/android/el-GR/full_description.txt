<b>Απλότητα</b>
Η διεπαφή είναι φιλική προς το χρήστη, και οι προσβολές είναι όλες υψηλής ποιότητας. Η εφαρμογή αναπτύχθηκε για να σας παρέχει μια ευχάριστη εμπειρία.
Στην κύρια οθόνη της εφαρμογής, θα έχετε τη δυνατότητα να δημιουργήσετε και να περιηγηθείτε σε διάφορες κακές προσβολές. Η χρήση της εφαρμογής δεν θα μπορούσε να είναι ευκολότερη και χρειάζεται μόνο ένα δευτερόλεπτο για να δημιουργηθούν νέες προσβολές. Μπορείτε να τα περιηγηθείτε με ένα απλό πάτημα της οθόνης σας.

<b>Δωρεάν & Ανοικτού Κώδικα</b>
Αυτή η εφαρμογή είναι εντελώς δωρεάν για χρήση και θα μείνει με αυτόν τον τρόπο για πάντα. Ο πηγαίος κώδικας είναι διαθέσιμος στο GitHub. Δεν χρειάζεται να ανησυχείτε για ειδικές συμμετοχές, ετήσιες συνδρομές, ή κρυφό κόστος για να δημιουργήσετε αποτελέσματα και να τα μοιραστείτε με άλλους.

<b>Κοινή χρήση</b>
Προσπαθήσαμε να απλοποιήσουμε τη διαδικασία χρήσης αυτής της εφαρμογής. Θα είστε σε θέση να μοιραστείτε προσβολές που παράγονται από το Evil Insult Generator χωρίς χρέωση. Οι προσβολές σας μπορούν να μοιραστούν με φίλους σε διάφορες εφαρμογές. 

<b>Τακτικά ενημερώθηκε</b>
Η ομάδα ανάπτυξης που δημιούργησε αυτή την εφαρμογή προσθέτει πάντα νέες προσβολές σε αυτήν. Κάθε προσβολή είναι εύκολο να μοιραστεί, και άλλοι μπορούν να τα χρησιμοποιήσουν, αν το επιθυμείτε. 
Χρειαζόμαστε όμως τη στήριξή σας. Παρακαλούμε να μας στείλετε τις προσβολές σας και θα το αναθεωρήσουμε. Αν μας αρέσει, η προσβολή θα είναι η νεότερη προσθήκη στην εφαρμογή.
